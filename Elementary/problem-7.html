Elementary - Problem #7

Original Problem

Write a program that prints a multiplication table for numbers up to 12.

Solution Demo

Click the button below to run the program.

Solution Code

Disclaimer: This solution may not be the best solution (by any means); it is simply being posted here as my solution. 

function genTable() {
    const tablediv = document.getElementById('problem');
    let table = document.createElement('table');
    let thead = document.createElement('thead');
    let tbody = document.createElement('tbody');
    let trow = document.createElement('tr');

    thead.appendChild(trow);
    table.appendChild(thead);
    table.appendChild(tbody);
    tablediv.appendChild(table);

    let varNew = [];
    let varNext = [];
    let varLast = [];
    let varS = [];
    
    for (let i = 0; i < 13; i++) {
        varNew[i] = document.createElement('th');
        trow.appendChild(varNew[i]);
        varNew[i].innerHTML = i;
    }
    
    for (let j = 1; j < 13; j++) {
        varNext[j] = document.createElement('tr');
        tbody.appendChild(varNext[j]);
        for (let k = 1; k < 2; k ++) {
            varLast[k] = document.createElement('td');
            varNext[j].appendChild(varLast[k]);
            varLast[k].innerHTML = j;
            varLast[k].classList.add('trowadd');
        }
        for (let l = 1; l < 2; l ++) {
            varS[l] = document.createElement('td');
            varNext[j].appendChild(varS[l]);
            varS[l].innerHTML = j * l;
            
            if ((j === l) && (l % 2 != 0)) {
                varS[l].classList.add('darkColor');
            } else if (j === l) {
                varS[l].classList.add('colored');
            }
        }
    }
}

The first chunk of code within the genTable() function sets us up for generating the table itself.

We create a <table> element, a <thead> element, and a <tbody> element, and append the <table> element to the already-existing div (tablediv), and the <thead> and <tbody> elements to the table. If these elements aren't appended correctly, this code won't work. 

let varNew = [];
let varNext = [];
let varLast = [];
let varS = [];

We set up four variables and initialize them as arrays so that we can perform for loops on them, which is how we're going to generate our 12 x 12 table. 

for (let i = 0; i < 13; i++) {
    varNew[i] = document.createElement('th');
    trow.appendChild(varNew[i]);
    varNew[i].innerHTML = i;
}

Ah, our first for loop! This loop will create the rows of our table.

  • We set the length of the loop as i < 13; because we have 13 rows total in our table.
  • varNew[i] = document.createElement('th'); gives us our header row, and trow.appendChild(varNew[i]); appends this row to the top of our table.
  • Finally, varNew[i].innerHTML = i;} populates the cells in the first row with our numbers (0 - 12).

This brings us to our next for loop, which consists of 2 additional for loops nested into it. Nesting loops into loops is like saying "For every iteration of loop #1, go through loop #2, and for every iteration of loop #2, execute code".
