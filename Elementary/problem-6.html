Elementary - Problem #6

Original Problem

Write a program that asks the user for a number n and gives them the possibility to choose between computing the sum and computing the product of 1, …, n.

Solution Demo

Click the button below to run the program.

Enter a number and choose whether you'd like the calculator to add or multiply numbers 1 - n.

Result:

Solution Code

Disclaimer: This solution may not be the best solution (by any means); it is simply being posted here as my solution. 

const addBtn = document.getElementById('addbtn');
const multiplyBtn = document.getElementById('multiplybtn');
var result = document.getElementById('problem-result');
var numberArr = [];
var newNum;
var sum = 0;
var total = 1;

addBtn.addEventListener('click', function () {
    let userInput = parseInt(document.getElementById('userInput').value, "0");
    for (let i =   0; i < userInput; i ++) {
        newNum = parseInt(userInput) - i;
        numberArr.push(newNum);
    }

    for (let j = 0; j < userInput.length; j ++) {
        sum += numberArr[j];
    }

    result.textContent = sum;
    document.getElementById('userinput').value = '';
    sum = 0;
    numberArr = [];
});

multiplyBtn.addEventListener('click', function() {
    let userInput = parseInt(document.getElementById('userinput').value, "0");
    for (let i = 0; i < userInput; i ++) {
        newNum = parseInt(userInput) - i;
        numberArr.push(newNum);
    }

    for (let j = 0; j < userInput.length; j ++) {
        total = total * numberArr[j];
    }

    result.textContent = total;
    document.getElementById('userInput').value = '';
    total = 1;
    numberArr = [];
});
