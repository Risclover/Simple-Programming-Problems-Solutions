Elementary - Problem #3

Original Problem

Modify the previous program such that only the users Alice and Bob are greeted with their names.

Solution Demo

Click the button below to run the program.

Solution Code

Disclaimer: This solution may not be the best solution (by any means); it is simply being posted here as my solution. 

function helloAliceBob() {
    let name = prompt("Hello! What's your name?");

    if (name === 'Alice' || name === 'Bob') {
        alert(`Hello, ${name}! How are you today?`);
    } else {
        alert(`Hello! How are you today?`);
    }
}

Notes/Comments

We're evolving Problem #2. This one requires the use of a condition. If the user's name is Alice or Bob, then we greet them by name; otherwise, we don't.

Otherwise, this problem is identical to Problem #2.
