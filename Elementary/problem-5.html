Elementary - Problem #5

Original Problem

Modify Program #4 such that only multiples of three or five are considered in the sum, e.g. 3, 5, 6, 9, 10, 12, 15 for n = 17

Solution Demo

Click the button below to run the program.

Solution Code

Disclaimer: This solution may not be the best solution (by any means); it is simply being posted here as my solution. 

function numberSum () {
    let userNumber = prompt("Please enter a number.");
    let numberArr = [];
    let newNum;
    let sum = 0;

    for (let i = 0; i < userNumber; i++) {
        newNum = parseInt(userNumber) - i;
        if (newNum % 3 === 0 || newNum % 5 === 0 || newNum % 15 === 0) {
            numberArr.push(newNum);
        }
    }

    for (let j = 0; j < numberArr.length; j++) {
        sum += numberArr[j];
    }

    alert(`The sum of the numbers from 1 to ${userNumber} is ${sum}.`);
}

Notes/Comments

This problem was surprisingly very easy! Since this is just an evolution on Problem #4, I copied and pasted the code from that problem into this one. Then, it was just matter of adding a quick if/else condition to the first loop to filter out anything but multiples of 3 and 5.

I also had to make sure that the numberArr.push(newNum); part was now inside of the if/else condition so that the code reads "If the number is divisible by 3 or 5, then push the number into the array; otherwise, do nothing."

Note: The % operator is the modulo operator. Instead of dividing two numbers, it gives the remainder after the division. Take this block of code for example: 

if (newNum % 3 === 0 || newNum % 5 === 0 || newNum % 15 === 0) {
    numberArr.push(newNum);
}

The newNum % 3 === 0 part means that the number is divided by 3 and the result has a remainder of 0, which would mean that the number is divisible by 3.
