Elementary - Problem #2

Original Problem

Write a program that asks the user for their name and greets them with their name.

Update: Added a little piece that calls the user "stranger" if they don't enter their name.

Solution Demo

Click the button below to run the program.

Solution Code

Disclaimer: This solution may not be the best solution (by any means); it is simply being posted here as my solution. 

function userName() {
    let name = prompt("Hi there! What's your name?");
    alert(`Welcome, ${name}!`);
}

Updated solution:

function userName() {
    let name = prompt("Hi there! What's your name?");
    if (name === null || name === '' || name === ' ') {
        alert("Welcome, stranger!");
    } else {
        alert(`Welcome, ${name}!`)
    }
}

Notes/Comments

This problem is in the same vein as Problem #1 - this, too, is also extremely easy. It does require a tad more knowledge, what with the addition that the user has to enter their name somewhere. This, of course, is a good job for the prompt function, which produces a pop up window with an input box for the user to enter something.

This also requires the developer to set the user's input to a variable, then send out a greeting with the user's input (their name) in the greeting.

Update

After testing out this program randomly one day and just pressing enter when prompted for my name, I noticed that it returned "Welcome, null", so I added the updated code to account for that.
