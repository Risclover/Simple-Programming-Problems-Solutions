Elementary - Problem #8

Original Problem

While the original problem says "Write a program that prints all prime numbers up to 100.", I decided to append this to "Write a program that prints all prime numbers up to the number entered by the user." instead, so that I can include an interactive aspect to this program.

Warning: Don't enter a number too high or it will either take a long time to compute, or it will crash your browser. (Trust me, I already tried it.)

Solution Code

Disclaimer: This solution may not be the best solution (by any means); it is simply being posted here as my solution. 

let userInput = document.getElementById('primeinput');

function isPrime(num) {
    for(var i = 2; i < num; i++)
        if(num % i === 0) return false;
    return num;
}

function primeNumbers() {
    let userNumber = parseInt(document.getElementById('primeinput').value);
    let primes = [];
    let appendResult = document.getElementById('problem');
    if (userNumber <= 1) {
        primes.push('Prime numbers begin at the number 2, you silly billy.');
    } else {
        for (let i = 2; i <= userNumber; i++) {
            if (isPrime(i)) {
                primes.push(' ' + i);
            }
        }
    }
    appendResult.textContent = primes;
}

const goBtn = document.getElementById('gobtn');
goBtn.addEventListener('click', primeNumbers);

userInput.addEventListener("keyup", function(event) {
    if (event.code === "Enter") {
        event.preventDefault();
        goBtn.click();
    }
});

Notes/Comments

Although I originally tried to create this via just one function, it wasn't working all that well, so I eventually decided to create two functions - one to check to see if a number is prime, and one to generate the array of prime numbers that shows after "Result".

I also decided to include a little bit near the end that allows the user to press the enter button instead of requiring them to press "Go" every single time. What can I say? I'm always about user experience. 

function isPrime(num) {
    for(var i = 2; i < num; i++)
        if(num % i === 0) return false;
    return num;
}

Here we have a basic "Check to see if a number is prime" function. In our for loop, we begin with i = 2; because prime numbers begin at the number 2, not 1.

The if(num % i === 0) return false; statement simply says "If the number divided by i returns a whole number, return false" (because if that's the case, it can't be prime, can it?). 

function primeNumbers() {
    let userNumber = parseInt(document.getElementById('primeinput').value);
    let primes = [];
    let appendResult = document.getElementById('problem');
    if (userNumber <= 1) {
        primes.push('Prime numbers begin at the number 2, you silly billy.');
    } else {
        for (let i = 2; i <= userNumber; i++) {
            if (isPrime(i)) {
                primes.push(' ' + i);
            }
        }
    }
    appendResult.textContent = primes;
}

Now we have our main function, primeNumbers, which is also quite simple. After the bit of code that decides to call the user a "silly billy" if they enter a number less than 2, the remaining code runs a for loop (again, starting at the number 2) that checks all numbers up to the user's number with the isPrime() function we described a moment ago. If it's prime, we push it into the primes array (along with a space for formatting purposes).

appendResult.textContent = primes; then occupies the 'Results' space with the results of the computation.
