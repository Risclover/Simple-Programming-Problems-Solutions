Lists & Strings - Problem #4

Original Problem

Write a function that returns the elements on odd positions in a list.

Solution Demo

Instructions: First create a list. Type each list item into the box below and click Submit (or press Enter) to add it to the list. When you're done, click the Return odd items button to generate a list of just the odd list items.

Keyboard Shortcuts

Enter = Submit

ctrl + z = Find odd items

ctrl + x = Reset

Your odd items:

Solution Code

Disclaimer: This solution may not be the best solution (by any means); it is simply being posted here as my solution. 

const submit = document.getElementById('submit');
const list = document.getElementById('listresults');
const reverse = document.getElementById('reverse');
const reset = document.getElementById('reset');
const yourList = document.getElementById('yourlist');
const yourOdds = document.getElementById('yourodds');
const oddResults = document.getElementById('oddresults');
let listItem;
let arr = [];

submit.addEventListener('click', arrayInput);
oddItems.addEventListener('click', returnOddItems);
reset.addEventListener('click', resetProgram);

function arrayInput() {
    yourList.innerHTML = "<strong>Your list:</strong>";
    const userInput = document.getElementById(userinput');
    if (userInput.value != '' && userInput.value != ' ') {
        arr.push(userInput.value);
        listItem = document.createElement('li');
        yourList.style.display = 'block';
        yourList.classList.add('ptop');
        listItem.textContent = userInput.value + ' ';
        list.appendChild(listItem);
        userInput.value = '';
    } else {
        return false;
    }
}

function returnOddItems() {
    oddResults.textContent = '';
    yourOdds.style.display = "block";

    for (let i = 0; i < arr.length; i++ {
        if (i% 2 === 0) {
            listItem = document.createElement('li');
            listItem.textContent = (i + 1) + '. ' + arr[i];
            oddResults.appendChild(listItem);
            oddResults.style.listStyle = "none";
        }
    }
}


function resetProgram() {
    location.reload();
}

const userInput = document.getElementById('userinput');

userInput.addEventListener("keyup", function(event) {
    if (event.code === "Enter") {
        event.preventDefault();
        submit.click();
    }
});

userInput.addEventListener("keyup", function(event) {
    if (event.ctrlKey && event.key === "z") {
        event.preventDefault();
        reverse.click();
    }
});

userInput.addEventListener("keyup", function(event) {
    if (event.ctrlKey && event.key === "x") {
        event.preventDefault();
        reset.click();
    }
});

Notes/Comments

I decided to just do a regular solution to this problem (rather than making a program that the user can actually interact with) because, if you think about it, doing that would require too much work for how simple the problem is. I'd have to allow the user to create a list, then have a second input element for them to enter the thing they want to see is in their list (which is dumb because in this instance, they can already see their list), and yeah... I'm not into it. The next problems will continue to have the user experience though.

Anyway, the solution to this problem is very simple. All you have to do is use the .find() method, which searches an array for the search term. If it isn't found, it returns undefined, which is why, in my solution, I have the function return true if the function doesn't return undefined.
